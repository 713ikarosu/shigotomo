## レスポンシブブレークポイント

| デバイス | 条件 |
|---------|------|
| デスクトップ | 769px以上 |
| タブレット | 768px以下 |
| モバイル | 480px以下 |

---

## モバイル対応の要点

- ハンバーガーメニュー（アニメーション付き）
- タッチフレンドリーなボタンサイズ（最低 44×44px）
- フルワイドCTAボタン
- iOSズーム防止（`input { font-size: 16px }`）
- ヒーローの縦書き→横書きへの自動切り替え
