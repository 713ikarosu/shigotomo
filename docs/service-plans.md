# サービスプラン仕様

## ライトプラン（現在ローンチ済み）

### コアコンセプト
「毎日の学習習慣を途切れさせないための"軽い伴走"」
本気で頑張る人へ、生活に無理なく学習を組み込むためのサポート。

### 価格
**月額 2,980円（税込）**

### 定員
**月15〜20名まで**（品質維持・満枠マーケティング）

### 提供サービス内容

| # | 内容 | 詳細 |
|---|------|------|
| ① | 初回ヒアリング（1回） | Googleフォーム＋10〜15分のDM面談。目的・勉強時間・家庭/仕事状況の確認 |
| ② | 個別学習計画の作成（月1回） | 月次学習ロードマップ・週次タスク可視化・生活リズム考慮 |
| ③ | 毎日の報告フォーム | 1分でできる報告（実施内容・集中度・所要時間）・週次進捗グラフ |
| ④ | 週次チェック（週1回） | 達成度コメント（短文）・翌週の改善ポイント1つ提示 |
| ⑤ | 質問受付（週3まで） | 学習方法・計画・モチベ維持のみ対象 |
| ⑥ | モチベ維持の仕組み | 月1回の小テスト・習慣化コツの共有 |
| ⑦ | プレミアム候補選定 | 1ヶ月継続者の中からプレミアムへ招待（定員制・こちらが選ぶ方式） |

### ライトでできないこと（NG）

| NG内容 | 理由 |
|--------|------|
| 問題の解説・添削・専門指導 | プレミアム向け |
| 毎日フィードバック | 週次まとめのみ（自走支援） |
| 進路相談・キャリアコーチング | プレミアム向け |
| スケジュールの毎日更新 | 月次のみ |
| 電話・ビデオ通話 | ライトでは一切なし |

## プレミアムプラン（構想段階・招待制）

- **対象**: ライトで一定期間継続できた人のみ招待
- **価格**: 月数万円想定（高額）
- **内容**:
  - 毎日フィードバック
  - 週1回の面談
  - 完全オーダーメイド学習計画
  - キャリア・生活習慣まで含めた伴走支援

## 親向け教育・子育てコンサル（別軸サービス・構想段階）

- オンライン相談（15〜60分）
- 子どもの勉強習慣・生活リズム・スマホ管理・メンタルへのアドバイス
- 医師×教育の視点が強み
